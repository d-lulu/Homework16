## 🔧 Реализованные функции

### 1. `double* full_elements(double *ptr_array, int n)`
- **Назначение**: Заполнение массива значениями функции из лаб. 8
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: указатель на заполненный массив

### 2. `int put_elements(double* ptr_array, int n)`
- **Назначение**: Печать элементов массива
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: количество выведенных элементов

### 3. `double* calc_elements(double* ptr_array, int n)`
- **Назначение**: Обработка элементов массива согласно индивидуальному варианту
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: указатель на обработанный массив

### 4. `double sum_elements(double* ptr_array, int begin, int end)`
- **Назначение**: Вычисление суммы элементов массива в диапазоне [begin, end]
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `begin` - начальный индекс (включительно)
  - `end` - конечный индекс (включительно)
- **Возвращает**: сумму элементов

### 5. `int find_element(double* ptr_array, int n, double element)`
- **Назначение**: Поиск элемента в массиве
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
  - `element` - искомый элемент
- **Возвращает**: индекс найденного элемента или -1 если не найден

### 6. `double find_max(double* ptr_array, int n, double a)`
- **Назначение**: Поиск в массиве максимального по модулю элемента, не равного заданному А
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
  - `A` - искомый элемент А
- **Возвращает**: максимальный по модулю элемент, не равный А, или -1 если не найден

### 7. `int delete_k(double* ptr_array, int n)`
- **Назначение**: Удаление всех элементов меньших среднего арифметического
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: число элементов массива без учета удаленных

### 8. `double* insert(double* ptr_array, int* n, int num)`
- **Назначение**: Вставка -999 перед первым отрицательным
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
  - `num` - вставляемое значение
- **Возвращает**: указатель на обработанный массив

### 9. `double* full_elements_1(double* ptr_array, int n, int a, int b)`
- **Назначение**: Заполнение массива случаными вещественными числами из указанного диапазона
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
  - `a` - начало диапазона
  - `b` - конец диапазона
- **Возвращает**: указатель на заполненный массив

### 10. `int max_size(int a, int b, int c)`
- **Назначение**: Поиск максимального размера трех массивов
- **Параметры**: 
  - `a` - размер первого массива
  - `b` - размер второго массива
  - `c` - размер третьего массива
- **Возвращает**: максимальный размер

### 11. `double find_min(double a, double b, double c)`
- **Назначение**: Поиск минимального из трех элементов
- **Параметры**: 
  - `a` - первый элемент
  - `b` - второй элемент
  - `c` - третий элемент
- **Возвращает**: минимальный элемент

### 12. `double* full_elements_d(double* ptr_array_a, int a, double* ptr_array_b, int b, double* ptr_array_c, int c, double* ptr_array_d, int d)`
- **Назначение**: Заполнение массива array_d по принципу array_d[i] = min(array_a[i], array_b[i], array_c[i])
- **Параметры**: 
  - `ptr_array_a` - указатель на массив array_a
  - `a` - размер массива array_a
  - `ptr_array_b` - указатель на массив array_b
  - `b` - размер массива array_b
  - `ptr_array_c` - указатель на массив array_c
  - `c` - размер массива array_c
- **Возвращает**: указатель на массив array_d
